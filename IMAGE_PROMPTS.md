# Memory Game — Image Generation Prompts

## How to use

1. Copy the **Style Lock** prompt first and send it to ChatGPT/DALL·E 3
2. Then send each card prompt one by one (they all inherit the style)
3. Save images as PNG, 256×256 or 512×512
4. Name files to match the `pairId` values below
5. Place in `static/images/{theme}/` (e.g. `static/images/dinosaur/dino-trex.png`)
